### What I learned

I have always neglected font family but i have got to know the usefulness and I know that the font face is used to import a font into your website. The code for that is below:

```css
@font-face {
    font-family: "Figtree font";
    src: url(./assets/fonts/Figtree-VariableFont_wght.ttf) format("truetype"), url(./assets/fonts/Figtree-VariableFont_wght.woff) format("woff");
}
body {
    background-color: hsl(47, 88%, 63%);
    font-family: "Figtree font", sans-serif;
}
```
